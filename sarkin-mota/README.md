# Sarkin Mota 2.0

A modern automobile marketplace and inventory management frontend prototype.

This project demonstrates a scalable UI system for vehicle browsing, filtering, comparison, and admin inventory management.

---

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- React Router
- Zustand
- Framer Motion

---

## Features (Current)

### Public Interface
- Vehicle listing
- Basic filtering-ready structure
- Vehicle pricing display
- Responsive layout foundation

### System Architecture
- Modular folder structure
- Centralized state management (Zustand)
- Mock data system
- Layout-based routing
